## Work

A list of projects I've built in my free time.

### OpenAPI Review Action

[Link](https://github.com/marketplace/actions/openapi-review)

A GitHub Action for automatically posting a PR comment with an
[openapi-diff](https://bitbucket.org/atlassian/openapi-diff) and
[widdershins](https://github.com/Mermade/widdershins) docs when a developer modifies an OpenAPI spec
file.

### Generate Windows Terminal Scheme

[Link](https://marketplace.visualstudio.com/items?itemName=blake-mealey.generate-wt-scheme)

A VSCode extension for generating a Windows Terminal colour scheme from the currently active VSCode
colour theme.

### Azure Commit Copy

[Link](https://chrome.google.com/webstore/detail/azure-commit-copy/cdpgnoklhcbaafbbemfglebpddbnedcp?hl=en)

A Chrome extension for quickly copying Git commit info from Azure Repos, then formatting it using
ES6 template literal syntax, and copying it to the user's clipboard.
